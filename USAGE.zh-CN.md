# Resend Client 使用指南

## 快速开始

### 1. 获取 Resend API Key

1. 访问 https://resend.com 并注册账号
2. 登录后进入 Dashboard
3. 点击 "API Keys" 创建新的 API Key
4. 复制生成的 API Key（格式：`re_xxxxxxxxxx`）

### 2. 验证发件域名

在 Resend 控制台：
1. 进入 "Domains" 页面
2. 添加你的域名（例如：`yourdomain.com`）
3. 按照提示添加 DNS 记录
4. 等待验证通过

### 3. 配置应用

1. 启动 Resend Client 应用
2. 点击左侧导航栏的「设置」
3. 输入你的 Resend API Key
4. 点击「保存 API Key」

## 发送邮件

### 基本步骤

1. 点击「撰写邮件」
2. 填写必填字段：
   - **发件人**: 必须使用已验证的域名邮箱（如 `noreply@yourdomain.com`）
   - **收件人**: 可以填写多个，用逗号分隔
   - **主题**: 邮件标题
3. 填写邮件内容（至少填写一个）：
   - **HTML 内容**: 支持 HTML 格式的邮件
   - **纯文本内容**: 纯文本格式的邮件
4. 点击「发送邮件」

### 示例

#### HTML 邮件示例

```html
<h1>欢迎使用我们的服务！</h1>
<p>感谢您的注册。</p>
<p>请点击下面的链接激活账号：</p>
<a href="https://example.com/activate">激活账号</a>
```

#### 纯文本邮件示例

```
欢迎使用我们的服务！

感谢您的注册。
请访问以下链接激活账号：
https://example.com/activate
```

## 查看历史

1. 点击「发送历史」查看所有已发送的邮件
2. 每条记录显示：
   - 邮件主题
   - 发件人和收件人
   - 发送时间
   - Resend ID（用于追踪）
   - 发送状态

## 常见问题

### Q: 为什么发送失败？

**A:** 可能的原因：
- API Key 未配置或无效
- 发件人邮箱未使用已验证的域名
- 收件人邮箱格式错误
- 网络连接问题

### Q: 数据存储在哪里？

**A:** 所有数据加密存储在本地：
- Windows: `C:\Users\你的用户名\.resend-client\config.enc`
- macOS: `/Users/你的用户名/.resend-client/config.enc`
- Linux: `/home/你的用户名/.resend-client/config.enc`

### Q: API Key 安全吗？

**A:** 是的，API Key 使用 AES-256-GCM 加密算法存储在本地，不会上传到任何服务器。

### Q: 可以发送附件吗？

**A:** 当前版本暂不支持附件，未来版本会添加此功能。

### Q: 发送限制是多少？

**A:** 发送限制取决于你的 Resend 账号套餐：
- 免费版：每月 3,000 封
- 付费版：根据套餐不同

## 技巧

1. **批量发送**: 在收件人字段用逗号分隔多个邮箱地址
2. **保存模板**: 可以将常用的 HTML 模板保存在本地文件中，需要时复制粘贴
3. **测试邮件**: 建议先发送测试邮件到自己的邮箱，确认格式正确
